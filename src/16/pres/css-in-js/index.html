Создание компонента

      import styled from "styled-components"
       
      const Input = styled.input `
          padding: 0.5em;
          margin: 0.5em;
          color: palevioletred;
          background: white;
          border: none;
      `;

      render(
          <div>
            <Input placeholder="Сопротивление бесполезно" />

          </div>
      );

Динамические свойства и интерполяция

          background: ${props => props.primary ? 'palevioletred' : 'white'} ;

          margin: 1em;
          padding: 0.25em 1em;
          border: 2px solid palevioletred;
          border-radius: 3px;

Инструменты

  • Линтинг - stylelint-processor-styled-components
  • Подсветка синтаксиса - vscode-styled-components
  • Тестирование состояний - jest-styled-components
  • Миксины и хелперы - polished ${ ✨ }
  • Автодополнение - Расширение css-in-js для Atom и VS code

Достоинства

  • Помогает мыслить компонентами
  • Поддержка тем из коробки
  • Полная изоляция стилей
  • Изолированные селекторы (по умолчанию)
  • Префиксы (по умолчанию)
  • Выделение критического CSS - стилей необходимых для отображения данного конкретного состояния приложения
  • Отложенное вычисление - стили создаются, только при подключении компонента
  • Удобное использование единого кода в CSS и JS - констант и функций
  • Умная оптимизация с Javascript: удаление кода, который не используется
  • Юнит тесты для CSS
  • Возможность переиспользования стилей на разных платформах: React Native

Недостатки

  • В 0,2% браузеров отключен Javascript
  • Инструменты всё ещё не идеальны: автодополнение не поддерживает строковые литералы
  • Читаемые классы для автоматизации тестирования
  • Не все знают Javascript
  • Пользовательский ввод нужно проверять: - Как использовать CSS-in-JS безопасно?
  • Отсутствие атомарного подхода к дизайну приводит к спагетти-коду: Атомарный веб-дизайн

      .btn {
          display: inline-block;      
          border: none;
          border-radius: $base-radius;
          cursor: pointer;
          font-size: $base-font-size;
          padding: 0 12px;
          /* больше стилей для обычной кнопки... */
       
          &--secondary {
              background: $secondary;
          }
       
          &--small {
              font-size: ($base-font-size * 0.875);
              line-height: 2.2;
              padding: 0 12px;
          }
          /* больше стилей для других вариаций кнопки... */
      };
