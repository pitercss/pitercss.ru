Задачи

  1. Общая
    • Всё должно быть одинаково для всех проектов
  2. Инструментальная
    • Инструмент пересборки внешнего вида
    • Лёгкая разработка внешних экранов
  3. Перспективная
    • Быстрая глобальная замена

Решение

  1. Договориваемся на берегу
    • Сетка и вертикальный ритм
    • Типографика
    • Цвета
  2. Выбрать набор компонентов
  3. Придумать механизм оформления элементов в нашем стиле

Практика

  1. Angular Material
    • Flexbox, задаваемый через атрибуты
      layout="column" layout-align="center start"
    • Адаптив

    • Биндиги к ангуляру
    • Компоненты
  • элементы страницы(меню, тултипы, табы, дропдауны)

Результат

  1. Фунциональные директивы
  2. Компоненты имеют заранее установленный вид
  3. Компоновка экранов, форм, списков - очень просто
  4. Настраиваемые через JSON темы в каждом проекте
    + $mdThemingProvider

  1. От нас ушёл дизайнер
  2. Для сайтов не подойдёт
  3. Нет анимаций
