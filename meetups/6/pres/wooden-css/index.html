Что в итоге?

Nest CSS

github.com/m18ru/nest-css

Правила

  • Небольшой элемент = файл, большой = директория
  • Имена — селектор
  • Соединяем через «>», для « » — префикс «!» (!a.css)
  • Префикс «_» — директории для группировки, файлы — не включаются
